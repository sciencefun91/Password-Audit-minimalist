1. Password Compromise Check

Check if your password has been exposed in data breaches using the free Pwned Passwords API from Have I Been Pwned.

How it works (k-anonymity model):

  • Your password is converted to a SHA-1 hash
  • Only the first 5 characters of the hash are sent to the API
  • The API returns all hashes starting with those 5 characters
  • Your browser compares the full hash locally
  • Your password never leaves your browser in plain text

The database contains over 600 million compromised passwords from real data breaches.
